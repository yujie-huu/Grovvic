Grovvic is a climate-smart gardening platform aligned with the UN SDGs. It helps users discover sustainable planting tips, explore seasonal recommendations, and learn practical sustainability knowledge through interactive content and simple simulations.

## Features
- Plant recommendation, search and detailed planting guide
- Weather forcast, trends, and relevant gardening tips
- Sustainability education content with quizzes
- Garden planting simulation (interactive and informative)
- Clear navigation and responsive UI

## Tech Stack
- **Frontend:** React (Vercel)
- **Backend:** Python + FastAPI (AWS)
- **Process:** Iterative development with usability feedback

## Live
- https://netvivo-grovvic.vercel.app/

## Getting Started (Local)
1. `npm install`
2. Create `.env` if needed (e.g. API base URL)
3. `npm run dev` and open the printed local URL

## Deployment
- Frontend deployed on **Vercel**
- Backend API hosted on **AWS**
